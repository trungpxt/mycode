giả ngố java

java là một ngôn ngữ lập trình hướng đối tượng, dựa trên lớp được thiết kế để có càng ít phụ thuộc thực thi càng tốt. Nó là ngôn ngữ lập trình có mục đích chung cho phép các nhà phát triển ứng dụng viết một lần, chạy ở mọi nơi (WORA), nghĩa là mã Java đã biên dịch có thể chạy trên tất cả các nền tảng hỗ trợ Java mà không cần biên dịch lại. Các ứng dụng Java thường được biên dịch thành bytecode có thể chạy trên bất kỳ máy ảo Java (JVM) nào bất kể kiến trúc máy tính nào. Nguyên nhân khiến chúng ta phải tải java vì minecraft không được biên dịch sẵn (còn tại sao người ta ko biên dịch thì ai mà biết đươc =} )

Những nói chung thì nếu ko có trình biên dịch java thì không thể chơi minecraft vì minecraft ko được biên dịch săn cho người chơi nên nếu ở đâu có clip hay trang web nào nói tải chơi minecraft mà ko cầm trình dịch java thì các bạn tốt nhất bỏ qua cho đỡ tốn thời gian.
